# Mobile Catalogue App

Bu proje, Flutter ve Dart üzerine alınan eğitimlerin çıktılarını tartmak ve göstermek üzere tasarlanmıştır.
Projenin amacı, kullanıcıların bir ürün kataloğunu görüp, ürünleri detay sayfalarında inceleyebildiği ve sepete ekleyip çıkarabildikleri bir uygulama ortaya koymaktır.

# Projede Kullanılanlar
## Geliştirme Araçları
- VS Code
- Flutter SDK
- Dart SDK
- Android Studio
- Android Emulator
- API için https://fakestoreapi.com

# İçindekiler
- Ürünlerin kataloglanıp gösterildiği bir ana sayfa
- Ürün detaylarının gösterildiği, ürün kartlarına tıklandığında aktif hale gelen ürün sayfaları
- İçinde ürün olmadığında boş gözüken bir sepet sayfası
- İçine ürün eklendiğide değişen, ürün eklenip çıkarılabilen bir sepet sayfası
